# Project 01: Frontend-First Full-Stack App

## Goal

Build a small product that proves frontend execution and basic full-stack understanding.

## Suggested Problem

Create a simple dashboard app with:

- sign in or mock auth flow
- list page
- create form
- edit flow
- validation
- loading, empty, and error states

Examples:

- task tracker
- habit tracker
- notes dashboard
- expense tracker

## MVP Scope

- one main entity
- one form
- one list or dashboard view
- one detail or edit flow
- basic auth or protected view
- deployment notes

## Suggested Stack

- frontend-first UI
- JavaScript or TypeScript
- API integration
- simple database later if needed

## Evidence Checklist

- screenshot or demo link
- README with setup steps
- feature list
- bug fixes logged
- lessons learned
- one short case study

## Weekly Milestones

### Week 1

- choose the problem
- define screens
- build static UI

### Week 2

- connect form and list flow
- add validation
- add loading and error states

### Week 3

- connect to API or mock backend
- improve empty states and polish
- write README

### Week 4

- clean up code
- document decisions
- add project evidence to profile builder
