# Projects

Keep one markdown page per portfolio project or practice app.

Recommended file naming:

- `Project-01-...`
- `Project-02-...`
- `Project-03-...`

Each project page should answer:

- what problem am I solving
- what is the smallest useful scope
- what stack am I using
- what did I personally build
- what evidence can I show later
