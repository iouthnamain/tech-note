# Developer Career OS

This folder is a personal markdown system for learning software engineering, building projects, improving execution quality, and turning that work into a strong public profile.

Use it as an operating system, not as a document archive:

- read from the roadmap
- build from the dashboard
- record evidence in project pages
- review progress weekly
- update your profile from real work

## Rules

- English first. Add Vietnamese only when it helps you think faster.
- Reuse repo guides instead of rewriting them here.
- Every topic ends with an artifact: note, code, bug write-up, demo, or shipped feature.
- Use AI to accelerate learning, but verify code and behavior yourself.
- Update this folder every week, not only when you feel motivated.
