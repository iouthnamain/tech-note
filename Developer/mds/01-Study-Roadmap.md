# Study Roadmap

This roadmap turns the existing repo into a practical path from beginner to job-ready frontend-first full-stack engineer.

## Phase 1: Foundations

**Duration:** 6 weeks

**Focus**

- JavaScript
- HTTP and request flow
- Git basics
- SQL basics
- HTML and CSS
- validation
- CRUD
- auth basics

**Read first**

- [Group 01: Foundation Review](../../Full-Stack-Guides/Group-01-Foundation-Review/README.md)
- [01.05 HTTP & RESTful API](../../Full-Stack-Guides/Group-01-Foundation-Review/01.05_HTTP_Protocol_RESTful_API.md)
- [01.10 JavaScript ES6+](../../Full-Stack-Guides/Group-01-Foundation-Review/01.10_JavaScript_ES6_Modern_JavaScript.md)
- [01.11 Database Basics](../../Full-Stack-Guides/Group-01-Foundation-Review/01.11_Database_Basics_SQL_Queries.md)
- [01.14 Error Handling](../../Full-Stack-Guides/Group-01-Foundation-Review/01.14_Error_Handling_Basics.md)
- [Group 02: Basic Functions](../../Full-Stack-Guides/Group-02-Basic-Functions/README.md)
- [02.01 CRUD](../../Full-Stack-Guides/Group-02-Basic-Functions/02.01_CRUD_Basic_Create_Read_Update_Delete.md)
- [02.02 Validation](../../Full-Stack-Guides/Group-02-Basic-Functions/02.02_Validation_Client_Server_Side.md)
- [02.08 Authentication](../../Full-Stack-Guides/Group-02-Basic-Functions/02.08_Authentication_JWT_Session.md)

**Checkpoint**

- You can explain how a form becomes a request, how the backend validates it, and how data reaches the database.

## Default Stack Path

Stay with one momentum path until you have shipped at least two projects:

- frontend-first full-stack
- modern JavaScript or TypeScript
- relational database
- simple deployment flow

Do not switch stacks often. Depth beats novelty at this stage.
