# Group 06: Database Analysis & SQL Optimization / NHÓM 6: Phân tích Database & Tối ưu SQL

## Overview / Tổng quan

**English**: This group focuses on database design, SQL optimization, and understanding when to use SQL vs NoSQL. Learn to analyze, design, and optimize databases effectively.

**Vietnamese**: Nhóm này tập trung vào thiết kế database, tối ưu hóa SQL và hiểu khi nào dùng SQL vs NoSQL. Học cách phân tích, thiết kế và tối ưu hóa database hiệu quả.
