## IDE Debuggers / Debugger IDE

### Example 1: VS Code Debugger / Ví dụ 1: Debugger VS Code

```json
// .vscode/launch.json
{
  "version": "0.2.0",
  "configurations": [
    {
      "type": "node",
      "request": "launch",
      "name": "Debug Jest Tests",
      "program": "${workspaceFolder}/node_modules/.bin/jest",
      "args": ["--runInBand", "--no-cache"],
      "console": "integratedTerminal",
      "internalConsoleOptions": "neverOpen"
    },
    {
      "type": "node",
      "request": "launch",
      "name": "Debug Current File",
      "program": "${file}",
      "skipFiles": ["<node_internals>/**"]
    }
  ]
}
```

## Browser DevTools / DevTools trình duyệt

### Example 2: Chrome DevTools / Ví dụ 2: Chrome DevTools

```typescript
// Using Chrome DevTools / Sử dụng Chrome DevTools
// 1. Open DevTools (F12)
// 2. Go to Sources tab
// 3. Set breakpoints
// 4. Use debugger statement

function processOrder(order: Order) {
  debugger; // Execution pauses here / Thực thi dừng ở đây
  const total = calculateTotal(order.items);
  return total;
}

// Console debugging / Debug console
console.log('Order:', order);
console.table(order.items);
console.group('Order Processing');
console.log('Step 1:', step1);
console.log('Step 2:', step2);
console.groupEnd();
```

## Command-Line Tools / Công cụ dòng lệnh

### Example 3: Node.js Debugger / Ví dụ 3: Debugger Node.js

```bash
# Node.js inspector / Node.js inspector
node --inspect app.js
# Connect with Chrome DevTools at chrome://inspect

# Python debugger / Python debugger
python -m pdb script.py

# Breakpoint in code / Breakpoint trong code
import pdb; pdb.set_trace()
```
