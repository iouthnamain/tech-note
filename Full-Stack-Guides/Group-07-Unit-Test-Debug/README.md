# Group 07: Unit Test, Debug, Fix Bug / NHÓM 7: Unit Test, Debug, Fix Bug

## Overview / Tổng quan

**English**: This group covers testing skills, debugging techniques, and bug lifecycle management as used in real companies. Learn to write tests, debug effectively, and manage bugs professionally.

**Vietnamese**: Nhóm này bao gồm kỹ năng testing, kỹ thuật debug và quản lý vòng đời bug như trong công ty thực tế. Học cách viết test, debug hiệu quả và quản lý bug chuyên nghiệp.
