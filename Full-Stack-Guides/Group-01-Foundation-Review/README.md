# Group 01: Foundation Review / NHÓM 1: Ôn tập nền tảng

## Overview / Tổng quan

**English**: This group provides the base concepts required for every later group in the folder. It is the right place to review HTTP, architecture, JavaScript, SQL, configuration, and error handling before moving into real full-stack implementation.

**Vietnamese**: Nhóm này cung cấp các khái niệm nền tảng cần thiết cho tất cả các nhóm phía sau trong thư mục. Đây là nơi phù hợp để ôn lại HTTP, kiến trúc, JavaScript, SQL, cấu hình và xử lý lỗi trước khi đi vào triển khai full-stack thực tế.

## Why This Group Matters / Tại sao nhóm này quan trọng

This group answers the foundational questions behind later implementation work:

- how requests move through a system
- how frontend and backend responsibilities differ
- how data should be modeled
- how code should be structured
- how runtime configuration should be handled
