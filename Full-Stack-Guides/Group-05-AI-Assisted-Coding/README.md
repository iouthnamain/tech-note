# Group 05: AI-Assisted Coding / NHÓM 5: Lập trình hỗ trợ AI

## Overview / Tổng quan

**English**: This group teaches how to use AI tools as a real engineering accelerator instead of a code generator with weak judgment. It covers prompting, debugging, code review, security checking, refactoring, documentation, and test generation.

**Vietnamese**: Nhóm này dạy cách dùng công cụ AI như một bộ tăng tốc kỹ thuật thực thụ thay vì chỉ là công cụ sinh code với khả năng phán đoán yếu. Nhóm bao gồm prompting, debugging, code review, security check, refactoring, documentation và test generation.

## Safe Usage Rules / Quy tắc sử dụng an toàn

- use AI to accelerate, not replace understanding
- require exact constraints in prompts
- verify generated code paths, especially auth and database logic
- review security and performance explicitly
- prefer code with tests, validation, and error handling
