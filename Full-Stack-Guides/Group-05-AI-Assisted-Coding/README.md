# Group 05: AI-Assisted Coding / NHÓM 5: VIBE CODING - Dùng AI hỗ trợ lập trình

## Overview / Tổng quan

**English**: This group focuses on using AI tools (ChatGPT, Copilot, etc.) to assist coding effectively and professionally. Learn prompt engineering, code generation, debugging, and best practices.

**Vietnamese**: Nhóm này tập trung vào sử dụng công cụ AI (ChatGPT, Copilot, v.v.) để hỗ trợ lập trình hiệu quả và chuyên nghiệp. Học kỹ thuật viết prompt, tạo code, debug, và thực hành tốt nhất.
