# Topic 1.1: Cấu trúc dữ liệu cơ bản (Array, List, Map, Set)
## Data Structures: Array, List, Map, Set

## Mô tả / Description

Ôn lại các cấu trúc dữ liệu cơ bản: Array, List (ArrayList, LinkedList), Map (HashMap, TreeMap), Set (HashSet, TreeSet). Hiểu rõ đặc điểm, thời gian truy cập, khi nào nên dùng cấu trúc nào. So sánh hiệu năng giữa các cấu trúc dữ liệu.

**English:** Review basic data structures: Array, List (ArrayList, LinkedList), Map (HashMap, TreeMap), Set (HashSet, TreeSet). Understand characteristics, access time, when to use each structure. Compare performance between data structures.

## Mục tiêu học tập / Learning Objectives

- Nắm vững đặc điểm và cách sử dụng từng cấu trúc dữ liệu
- Hiểu được độ phức tạp thời gian (time complexity) của các thao tác cơ bản
- Biết chọn đúng cấu trúc dữ liệu phù hợp với từng tình huống

**English:**
- Master characteristics and usage of each data structure
- Understand time complexity of basic operations
- Know how to choose the right data structure for each situation

## Kỹ năng đạt được / Skills Achieved

- Phân tích và lựa chọn cấu trúc dữ liệu tối ưu
- Viết code hiệu quả với cấu trúc dữ liệu phù hợp
- Debug các vấn đề liên quan đến cấu trúc dữ liệu

**English:**
- Analyze and select optimal data structures
- Write efficient code with appropriate data structures
- Debug data structure-related issues
