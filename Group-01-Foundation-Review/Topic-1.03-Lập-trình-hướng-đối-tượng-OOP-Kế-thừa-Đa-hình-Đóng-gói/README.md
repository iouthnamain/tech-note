# Topic 1.3: Lập trình hướng đối tượng (OOP) - Kế thừa, Đa hình, Đóng gói
## Object-Oriented Programming: Inheritance, Polymorphism, Encapsulation

## Mô tả / Description

Ôn lại 4 tính chất của OOP: Tính đóng gói (Encapsulation), Tính kế thừa (Inheritance), Tính đa hình (Polymorphism), Tính trừu tượng (Abstraction). Hiểu rõ cách áp dụng trong thực tế, các lỗi thường gặp và cách tránh.

**English:** Review 4 OOP principles: Encapsulation, Inheritance, Polymorphism, Abstraction. Understand how to apply them in practice, common mistakes and how to avoid them.

## Mục tiêu học tập / Learning Objectives

- Nắm vững các khái niệm OOP cơ bản
- Áp dụng đúng các nguyên tắc OOP trong code
- Tránh các lỗi phổ biến khi sử dụng OOP

**English:**
- Master basic OOP concepts
- Apply OOP principles correctly in code
- Avoid common mistakes when using OOP

## Kỹ năng đạt được / Skills Achieved

- Thiết kế class và object đúng chuẩn
- Sử dụng kế thừa và đa hình hiệu quả
- Viết code dễ bảo trì và mở rộng

**English:**
- Design classes and objects correctly
- Use inheritance and polymorphism effectively
- Write maintainable and extensible code
