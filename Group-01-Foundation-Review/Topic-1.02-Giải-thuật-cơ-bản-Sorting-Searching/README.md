# Topic 1.2: Giải thuật cơ bản (Sorting, Searching)
## Algorithms: Sorting and Searching

## Mô tả / Description

Ôn lại các giải thuật sắp xếp phổ biến: Bubble Sort, Quick Sort, Merge Sort, Heap Sort. Các giải thuật tìm kiếm: Linear Search, Binary Search. So sánh độ phức tạp và ứng dụng thực tế của từng giải thuật.

**English:** Review common sorting algorithms: Bubble Sort, Quick Sort, Merge Sort, Heap Sort. Searching algorithms: Linear Search, Binary Search. Compare complexity and real-world applications of each algorithm.

## Mục tiêu học tập / Learning Objectives

- Hiểu rõ cách hoạt động của từng giải thuật
- Phân tích độ phức tạp thời gian và không gian
- Ứng dụng đúng giải thuật vào bài toán thực tế

**English:**
- Understand how each algorithm works
- Analyze time and space complexity
- Apply the right algorithm to real-world problems

## Kỹ năng đạt được / Skills Achieved

- Implement các giải thuật cơ bản
- Chọn giải thuật phù hợp cho từng bài toán
- Tối ưu hóa giải thuật khi cần thiết

**English:**
- Implement basic algorithms
- Choose the right algorithm for each problem
- Optimize algorithms when needed
