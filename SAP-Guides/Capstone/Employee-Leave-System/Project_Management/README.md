# Project Management Documentation

This folder contains all project management-related documentation for the Employee Leave Request and Approval System capstone project.

## Contents

- **[Project_Management_Plan.md](Project_Management_Plan.md)** - Comprehensive project management plan including:
  - Sprint structure (12-week, 1-week sprints)
  - Story point evaluation (Fibonacci scale)
  - Sprint planning process
  - Daily standup meetings
  - Sprint review and retrospective
  - Product backlog management
  - Velocity tracking
  - Burndown charts
  - Task management
  - Risk management
  - Metrics and KPIs
  - Communication plan

## Purpose

This documentation provides the framework for managing the capstone project using Agile/Scrum methodologies adapted for academic capstone projects. It includes:

- **Sprint Planning**: How to plan and execute 1-week sprints
- **Story Point Estimation**: Guidelines for estimating work using Fibonacci scale
- **Team Ceremonies**: Daily standups, sprint reviews, and retrospectives
- **Tracking**: Velocity, burndown charts, and metrics
- **Templates**: Ready-to-use templates for user stories, tasks, and ceremonies
