# Sprint 1: Requirements Gathering

**Dates**: January 5-9, 2026 (Monday-Friday)  
**Sprint Goal**: Gather and document all requirements for the Employee Leave Request and Approval System  
**Story Points Target**: 20-25 points  
**Phase**: Phase 1 - Requirements & Design

## Sprint Overview

### Sprint Goal
To comprehensively gather, analyze, and document all functional and non-functional requirements for the Employee Leave Request and Approval System, establishing a solid foundation for the design and development phases.

### Team Capacity
- **Team Size**: 5 members
- **Sprint Duration**: 5 days (Monday-Friday)
- **Individual Capacity**: ~35 hours per member
- **Total Team Capacity**: 175 hours
- **Available Capacity** (after meetings): ~132 hours
- **Story Points Capacity**: 20-25 points

### Key Deliverables
1. Requirements specification document
2. Data model requirements document
3. Workflow requirements document
4. UI/UX requirements document
5. Technical specification document
6. HR integration specification
