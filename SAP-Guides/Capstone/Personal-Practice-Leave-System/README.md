# Personal Practice Leave System - README

**12-week solo ABAP development project**

---

## 📚 Project Overview

This is a personal practice project to build an Employee Leave Request and Approval System using SAP ABAP. This project is adapted from a team-based capstone project to a solo 12-week learning journey.

---

## 🎯 Project Goals

- Learn ABAP development from basics to advanced
- Build a complete leave management system
- Practice real-world SAP development
- Create portfolio-worthy project
