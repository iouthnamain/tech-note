# RISK-FRD-002 - Data Accuracy & Reconciliation

## Description
Data discrepancies between FI/CO extracts and reported metrics could undermine trust in dashboards.

## Response Strategy
**Strategy**: Mitigate

**Response Plan**:
- Reconciliation scripts between source FI/CO tables and dashboard aggregates
- Variance checks and exception reports
- Data quality gates in extraction jobs

**Status**: Mitigated after recon scripts and variance checks were implemented.
