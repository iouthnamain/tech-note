# Frontend Guides

Thư mục này chứa các tài liệu hướng dẫn frontend, bao gồm khóa React nâng cao của Sydexa.

## Khóa học hiện có

- [Sydexa React Course](./sydexa-react-course/README.md)

## Mục tiêu

- Tập trung vào tài liệu học React nâng cao và Next.js
- Tổ chức theo tuần, mỗi tuần có một thư mục riêng
- Cung cấp tài liệu bằng tiếng Việt với nội dung chi tiết và ví dụ thực hành
